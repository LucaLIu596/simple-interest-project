# Simple Interest Shell Script Project
This open-source project provides a bash script named `simple-interest.sh` to calculate simple interest by accepting user input for principal, annual interest rate, and time in years. This repository follows standard open-source project specifications, including an Apache 2.0 license, contribution guidelines, and a community code of conduct.

## Repository File Structure
- `simple-interest.sh`: Executable shell script for simple interest calculation
- `LICENSE`: Complete Apache License 2.0 legal terms
- `CONTRIBUTING.md`: Step-by-step guide for new contributors
- `CODE_OF_CONDUCT.md`: Rules for respectful community collaboration
- `README.md`: Project overview, usage instructions and core information

## How to Run the Script Locally
1. Clone this repository to your local machine
2. Open your terminal and navigate to the project folder
3. Grant execution permission to the script:
```bash
chmod +x simple-interest.sh
Run the calculation program:
bash
运行
./simple-interest.sh
Follow the on-screen prompts to input principal, interest rate, and time to view the final result.
Calculation Formula
Simple Interest = (Principal × Annual Interest Rate × Time) ÷ 100
Total Payable Amount = Principal + Simple Interest
